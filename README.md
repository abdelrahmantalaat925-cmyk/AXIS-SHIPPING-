# AXIS-SHIPPING-
Static website ready to upload to GitHub and deploy on Vercel.

Files
index.html: main website page
styles.css: design and layout
script.js: editable contact details
assets/axis-logo.jfif: company logo
assets/Logistics_Proposal.pdf: proposal file shown inside the site
Edit before publishing
Open script.js and replace these placeholder values:

sales@axisshipping.com
+20 XXX XXX XXXX
Cairo, Egypt
201000000000
Deploy on GitHub and Vercel
Create a new GitHub repository.
Upload all files from this folder to the repository root.
Go to Vercel and choose Add New Project.
Import the GitHub repository.
Vercel will detect it as a static site automatically.
Click Deploy.
Notes
The proposal section displays the PDF directly on the site.
The design is optimized for desktop and mobile.
You can later add a real contact form or Arabic language version if needed.
